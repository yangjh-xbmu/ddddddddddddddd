# 2024news-2

## 地位
项目根目录，承载 HTML、CSS、教学材料与辅助脚本。

## 逻辑
当前目录下的静态页面与样式文件用于内容展示，Markdown 与文本文件用于教学和素材整理，git-auto-push.sh 用于自动化执行 Git 提交与推送。

## 约束
根目录脚本默认在 Git 仓库内运行。提交信息使用系统当前日期时间。无变更时脚本直接退出，不创建空提交。

| 子模块/文件 | 功能说明 |
|------------|---------|
| 0401.html | 页面内容文件 |
| main.css | 页面样式文件 |
| *.md | 教学文档与题库 |
| 0408.txt | 文本素材 |
| git-auto-push.sh | 自动执行 add、commit、push 的脚本 |
