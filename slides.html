JavaScript 概览

一种轻量级、解释型的编程语言，是 Web 的三大核心技术之一。

核心特性

  • 动态类型 — 变量无需声明类型，运行时自动推断
  • 事件驱动 — 基于回调与异步机制响应用户交互
  • 多范式 — 支持面向对象、函数式、命令式编程风格
  • 跨平台 — 浏览器、服务器 (Node.js)、移动端均可运行

应用领域

前端开发 后端 (Node.js) 移动应用 (React Native) 桌面应用 (Electron) 脚本自动化

语法速览

变量与函数

const name = "JavaScript";
let version = 2024;

function greet(who) {
  return `Hello, ${who}!`;
}

// 箭头函数
const add = (a, b) => a + b;

异步编程

async function fetchData() {
  const res = await fetch("/api");
  const data = await res.json();
  return data;
}

数组与对象

const langs = ["JS", "TS", "Py"];

langs.map(l => l.toLowerCase());
langs.filter(l => l !== "Py");

const obj = { name, version };
const { name: n } = obj; // 解构
